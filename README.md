# AgriData_Explorer_project
AgriData Explorer: Understanding Indian agriculture with EDA.
The provided data is downloaded as CSV file.
collect data and read the data by importing pandas as pd and save the data in a dataframe as df.
After that the df information and duplicates were checked , following that null values was also checked.
Though there were many columns, dropped the unwanted columns for the project given, first by displaying the columns and then analysed the wanted columns the dropped the unwanted column.
After the data cleaning process, a cleaned data is used for EDA.
Used statistical methods to identify insights given such as high and low production areas, top-performing crops, and yield etc.
additionally used ploty, numpy for better visualizations by different charts as barchart, pie chart, line plot, scatter plot etc.
Following that connected to sql using mysql connector and created table then inserted the values,
found queries for all the questions given as per project. 
After this installed powerBI and started creating a dasboard there and displayed all the sql queries as differnt charts for visualization.
